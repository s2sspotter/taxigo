TaxiGo

TaxiGo App is designed for taxi booking purpose and driver allocating. Drivers can login into this application and get proper access to avail this service and no need to pay any amount to use avail this service and no payment information also required.

Privacy policy

User information like Licence or M-Vahan is collected to verify whether the user is valid driver. This is completely free to use and collected information is stored securely in Cloud Storage and this information will be used only for verifying the user.
